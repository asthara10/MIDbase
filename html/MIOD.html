MIOD's purpose

The main goal of MicroIndels Open Database (MIOD) is to index all reported microdeletion and microinsertions into a single database. MIOD is an open database, intended for people without a genetic background interested in the possible effects of some microindel mutations.

MIOD's consortium

Beign part pf the MIOD consortium will allow you to submit data to our database. This function will only be available for people of the consortium to ensure that data is veridic. Members of the consortium should sign up using using a university/institute mail. Members will also have a place to deal any problems with their data.

How to use MIOD

In MIOD there are several tools you can use if you need them:

How to use the advanced search: If you go to the advanced search you can search for microindels in the database with different queries, like the name of the indel, the position, the gene they are related to, or its clinical significance among others.

How to introduce data: If you want to introduce data (and you are part of the MIOD consortium) you can introduce data following the form provided in the web.

How to contact us: If you have any problem with the data you submitted or any question from the MIOD project you can contact us using your email, in the contact window.
